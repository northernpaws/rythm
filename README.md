# Rythm

## Modules

### Mini

  * 480MHz ARM Cortex-M processor
  * 64MB of SDRAM
  * QSPI Flash
  * Stereo Audio Codec

### Card

Everything from the Mini:

  * 480MHz ARM Cortex-M processor
  * 64MB of SDRAM
  * QSPI Flash
  * Stereo Audio Codec
  * SDMMC 4-bit interface

Plus:

  * 1 Hardware MIDI Endpoint (RX and TX)
  * Eurorack x CV inputs
  * Eurorack x CV outputs
  * Eurorack Sync In/Out
